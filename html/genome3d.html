Genome3D Tutorials

These tutorials provide an introduction to Genome3D and related resources. It may help to start by completing Search and at least starting Genome3D Structures but the resource tutorials can be completed in any order.

We recommend using Firefox or Chrome rather than Internet Explorer for these tutorials.

General Tutorials

  • Search [~5 minutes]
    • Introduction to Genome3D's search abilities
  • Genome3D Structures [~15 minutes]
    • Introduction to examining and comparing Genome3D's 3D structural models

Genome3D Resource Tutorials

  • SUPERFAMILY [~10 minutes]
    • Introduction to some of the domain assignment and phylogenetic features of the SUPERFAMILY database.
  • CATH/Gene3D [~10 minutes]
    • Introduction to some of the features of the CATH/Gene3D resources.
  • PHYRE2 [~10 minutes]
    • Introduction to some of the features of PHYRE2.
  • FUGUE/VIVACE [~10 minutes]
    • Introduction to some of the features of the FUGUE/VIVACE resources.
  • DISOPRED, FFPred, MEMPACK [~10 minutes]
    • Introduction to some of the features of the DISOPRED/FFPred/MEMPACK resources.
  • SCOP [~10 minutes]
    • Introduction to some of the features of SCOP.
